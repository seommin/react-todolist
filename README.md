## react-todolist

패스트캠퍼스 프론트엔드 개발 with React Online 강의 예제

### `TodoTemplate`

Todo-List를 보여주기 위한 공간

### `TodoHead`

오늘 날짜 및 요일, 남은 할 일 개수 표시

### `TodoList`

완료한 일 또는 해야할 일 목록

### `TodoCreate`

할 일 추가를 위한 버튼 및 동작

### `TodoItem`

TodoList에 표시될 값들

### `TodoContext`

TodoItem 초기값 설정 및 react hook 동작
